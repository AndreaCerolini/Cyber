# Cyber

Questo materiale contiene quello che ho capito io sugli argomenti spiegati a lezione, messi insieme studiando appunti.
Non è una spiegazione ufficiale del professore, ma una rielaborazione personale fatta per rendere i concetti più chiari e ordinati.

In alcuni punti gli argomenti sono spiegati in modo più approfondito del necessario: in diversi casi il livello di dettaglio può andare oltre quello che probabilmente verrà chiesto nella verifica o all’esame. È una scelta voluta, per capire bene i concetti e non limitarsi a studiarli a memoria.
